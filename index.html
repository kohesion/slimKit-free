Don't tie your project down

  • no external dependencies (plain vanilla js, no jquery required)
  • only support two latest browser releases per vendor (ex. IE10 and IE11)
  • minified / obfuscated version available for free in public repository (coming soon)
  • CSS / image files optional
  • Simple Configuration - we avoid the Swiss Army knife approach
  • Created with modification in mind
  • SPA friendly - widgets are resilient enough to withstand DOM Mutation
